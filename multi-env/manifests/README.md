# manifests

## 예제 인덱스

| 디렉터리/파일 | 설명 |
|---|---|
| `applicationset-dev.yaml` | dev 환경 ApplicationSet (자동 동기화) |
| `applicationset-stage-prod.yaml` | stage, prod 환경 ApplicationSet (수동 동기화) |
| `apps/dev/` | dev 환경 nginx 배포 manifest |
| `apps/stage/` | stage 환경 nginx 배포 manifest |
| `apps/prod/` | prod 환경 nginx 배포 manifest |
